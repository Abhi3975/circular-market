# Changelog

All notable changes to the EcoCycle Circular Market project will be documented in this file.

## [1.8.2] - 2026-04-17
### Added
- **Asset Provenance:** Detailed lifecycle timeline for circular assets.
- **Impact Intelligence:** Material breakdown Donut charts (Recharts) and regional recovery map.
- **Enterprise Dashboard:** Multi-tab layout for Ledger and Geospatial analytics.
- **Command Palette:** Integrated Cmd+K support for rapid material discovery.

### Fixed
- **Routing:** Resolved 404 errors on deep-linked item pages via `404.html` fallback and `trailingSlash` optimization.
- **Persistence:** Corrected wallet balance hydration issues on initial load.

## [1.5.0] - 2026-04-13
### Added
- **Aesthetic Overhaul:** High-fidelity motion design, ambient cursor glows, and physics-based card interactions.
- **State Persistence:** LocalStorage integration for transaction history and inventory.

## [1.0.0] - 2026-04-02
- Initial release with marketplace feed and impact points system.
